# Screenshots & demo media

Drop your demo screenshots / photos here using the **exact filenames** referenced in the
main [`README.md`](../../README.md) so the images render automatically.

| Filename | What to capture |
|---|---|
| `01-app-connected.png`        | App home screen showing **Connected to RuralReach** status |
| `02-app-send-sms.png`         | App *Send SMS* screen filled in (number + message) |
| `03-sms-delivered-phone.png`  | Recipient's ordinary phone showing the **delivered SMS** |
| `04-reply-card.png`           | App home screen showing an **incoming reply** card |
| `05-node-oled.png`            | Node OLED showing a received reply / status |
| `06-voice-ptt.png`            | App *Voice Call* screen during push‑to‑talk |
| `07-idf-monitor-node.png`     | `idf.py monitor` log of the **node** (Codec2 self‑test, mic levels, TX) |
| `08-idf-monitor-gateway.png`  | `idf.py monitor` log of the **gateway** (LoRa RX + RSSI + SMS sent) |
| `09-hardware-bench.jpg`       | Photo of the assembled node + gateway hardware on the bench |
| `10-field-test.jpg`           | Photo of the field/dead‑zone test (optional) |

Photos may be `.jpg`; UI/log captures should be `.png`. Keep each under ~1 MB.
